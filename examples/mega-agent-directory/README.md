# MEGA Agent Directory Example

This directory contains example analysis artifacts from a specific domain dataset.

## Contents

- `schemas/`: captured table DDL
- `queries/`: exploratory SQL used during analysis
- `reports/`: generated markdown reports
- `reference-data/`: supporting JSON inputs used in the workflow
- `architecture.md`: domain-specific product context captured during analysis

These files are examples only. They are not the source of truth for the toolkit itself.
